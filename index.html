Quick Example

            
var rework = require('rework');
var plugin = require('plugin');
var plugin2 = require('plugin2');

var str = fs.readFileSync('mystylesheet.css', 'utf8');
var css = rework(str)
  .use(plugin())
  .use(plugin())
  .toString();

What about SASS/LESS

  • Lower level
  • No utilities (watch etc)
  • No logic built in (for, if etc)

Why Rework?

  • All Node stack
  • Easily extend the preprocessing to fit your needs
  • Invent your own properties!
  • Build your own "SASS"

How it works

  1. Create AST
    var css = rework(str)
  2. Pass AST to each plugin and let them modify the AST in turn 
    .use(plugin())
  3. Create string form AST — voila! 
    .toString();
